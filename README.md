# DBMS Practical Solutions

This repository contains solutions for various DBMS (Database Management Systems) practical problems, including:
- **MySQL**: DDL, DML, Joins, and Subqueries.
- **Oracle PL/SQL**: Triggers, Cursors, Procedures, and Functions.
- **MongoDB**: CRUD operations and Aggregation Framework.
